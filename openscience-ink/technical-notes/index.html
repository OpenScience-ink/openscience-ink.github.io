Technical notes

Crawlable public surface for OpenScience.ink

OpenScience.ink has a public sitemap index with topic, issue, and paper sitemaps. This GitHub Pages site documents that structure without duplicating the main domain's content.

Observed public structure

  • Root homepage, privacy, and terms pages.
  • Topic hubs with paper archives and filtered views.
  • Weekly issue pages across biomedical topics.
  • Individual paper pages for readable research paper summaries.

Why this GitHub Pages site exists

This site is a small public documentation surface. It gives search engines, AI answer engines, and directory reviewers a stable, crawlable explanation of what OpenScience.ink is and where the main biomedical research archives live.

Primary source of truth

The main domain remains the source of truth for current content, topic archives, issue pages, and summaries: https://openscience.ink.
